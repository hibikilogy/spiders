# Spiders
一系列根据各论坛 / 网站帖子 / 文章 ID 生成指定格式 markdown 文件的爬虫。自行修改格式可以做到快捷转载文章到使用 markdown 的博客。

目前功能较为完善的网站：
- 贴吧
- 虎扑
- bilibili

## 使用方法
运行 .py 文件，根据命令行提示输入帖子 / 文章 ID。

对于 Windows 系统，需要事先安装好 Python。
